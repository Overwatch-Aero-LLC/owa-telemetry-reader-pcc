Serial_t *  pPort 
)

Look for a gimbal packet arriving over a serial port.

Search for a gimbal packet coming over a physical serial channel. If a gimbal packet is found this function returns TRUE. The packet storage provided by the caller MUST persist between calls, since this function may partially complete a packet and then return FALSE while waiting for more data to arrive.

Parameters
pPktpoints to a packet structure which must persist between calls.
pPortidentifies a previously opened serial port.
Returns
TRUE if a packet is found, else FALSE.
