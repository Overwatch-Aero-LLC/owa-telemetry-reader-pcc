Detailed Description

Countdown timers for MPC555 systems.

The count down timers are based upon the 64-bit timebase counter of the MPC555. The counters are used by first calling WrTimer(), and then later calling RdTimer() to determine if the timer has elapsed.

Reset the caller stored countdown timer. The timeout value must not be more than 2^31 bits of the timebase. Hence at 1MHz the maximum timeout is 2147.5 seconds, or about 36 minutes. If longer timeout values are needed use WrTimer64() or WrTimerLong().
