float  StartFreq,

UInt16  Dwell,
UInt8  NumMeasurements

Request ground station spectrum sweep, coherent frequency radio.

Request ground station spectrum sweep. This puts the coherent frequency MHX radio in a scanning mode mesureing the signal strength at each user specified frequency. If the MXH radio in the groundstation is not coherent (i.e. it is hopping) then the user settings will not apply.

Parameters
pCommpoints to the communication interface data.
pPktpoints to a stream packet to receive the formatted data.
StartFreqis the frequency to begin the sweep in MHz.
Spacingis the space between frequencies in MHz.
Dwellis the dwell time for each measurement in ms.
NumMeasurementsis the number of measurements to make.
