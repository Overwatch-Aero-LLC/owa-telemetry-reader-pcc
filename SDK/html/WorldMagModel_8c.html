Detailed Description

The World Magnetic Model The World Magnetic Model calculates the Earth's magnetic field at a given geographic location and at a known date. The magnetic field is modeled as the negative gradient of a potential expressed in terms of spherical harmonics. The model is presented in detail at the National Geophysical Data Center's website: http://www.ngdc.noaa.gov/seg/WMM/DoDWMM.shtml.

Initialize the magnetic model persistent data.

Initialize the magnetic model persistent data which includes the gnm and hnm harmonic coefficients.

Parameters
Gnmis a compact single-precision array that will receive the Schmidt-normalized gnm(t) coefficients. The user must provide persistent storage for this array.
Hnmis a compact single-precision array that will receive the Schmidt-normalized hnm(t) coefficients. The user must provide persistent storage for this array.
Timeis the current date/time as a floating-point year (for example, July 1, 2005 = 2005.5)
