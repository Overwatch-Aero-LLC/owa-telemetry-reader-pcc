enum IridiumDefaultModes

Enumerate the Iridium modes.

Enumerator
HANGUP_OFF_MODE 

Hangup to an idle state.

HANGUP_MODE 

Hangup and resume dial or answer mode.

DIAL_MODE 

Start out in dial mode.

AUTOANSWER_MODE 

Start out in autoanswer mode.

CONNECTED_MODE 

Start out connected (and resume dial/answer mode as necessary)
