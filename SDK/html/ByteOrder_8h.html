Detailed Description

Network and host byte ordering functions.

This module contains the functions to switch between network and host byte order.

Function Documentation

double DataToDouble ( const UInt8 *  pData,
BOOL *  pInvalid 
)

Convert a string of bytes in network byte order to a double precision 64-bit floating point number in host byte order. Note that pData must point to an aligned address. If it does not use the function DataToDoubleMisaligned().

Parameters
pDatapoints to a four byte string of data in network byte order
pInvalidpoints to space to receive a flag indicating if the data represent a valid floating point number. If it does not then pInvalid will be written to a 1.
Returns
The floating point value encoded in the string.
