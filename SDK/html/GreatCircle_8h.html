Detailed Description

Great circle utilties.

The great circle module provides utilities for doing great circle calculations and transferring between LLA and ECEF formats

Endis an Earth-Centered-Earth-Fixed vector of the ending waypoint.
TrackPositionis an array to receive the X, Y, and Z (zero) track values.
TrackVel1is an array to receive the X, Y, and Z (zero) track relative velocities for Vel1.
TrackVel2is an array to receive the X, Y, and Z (zero) track relative velocities for Vel2. Pass NULL to not perform this calculation.
