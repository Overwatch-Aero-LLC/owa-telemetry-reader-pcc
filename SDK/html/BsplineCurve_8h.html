Detailed Description

3-D basis spline curve library.

This module provides functions for computing b-spline curves of arbitrary degree. It also includes functions for calculating the first and second-order derivatives, generation of knot vector distributions, global interpolation, global approximations, for 3-dimensional problems. The main references for the algorithms implemented here are http://www.cs.mtu.edu/~shene/COURSES/cs3621/NOTES/ web notes, and the textbook "A Practical Guide to Splines" by C. de Boor.

Calculate distance along the b-spline curve between two parameter values.

Calculate distance along the b-spline curve between two parameter values. This is done by integrating ds = sqrt((dx/du)^2+(dy/du)^2+(dz/du)^2)*du between the start and stop parameters.

Parameters
Q1is an array with the 3-D control points of the first-order derivative of the b-spline, size nx3.
ndefines the size of the control point array.
ucis the knot vector for the b-spline curve, size m+1.
mdefines the size of the knot vector.
ustartis the start parameter.
ustopis the stop parameter.
m1is the number of grid points from "ustart" to "ustop". Since the grid has a fixed number of points, the smaller the parameter interval, the more accurate the calculated distance will be.
Returns
The distance along the b-spline curve between the two parameter values.

Evaluate a bspline curve at a single point in the parameter domain.

De Boor's algorithm is a generalization of de Casteljau's algorithm. It provides a fast and numerically stable way for finding a point on a B-spline curve given a u in the domain. Reference: http://www.cs.mtu.edu/~shene/COURSES/cs3621/NOTES/spline/B-spline/de-Boor.html Note that the degree of the curve is implicitly calculated from the number of control points "n" and the number of knots "m" as p = m - n - 1.

Parameters
Pis the 3-D control point array, size (n+1)x3.
nspecifies the size of the control point array.
ucis the knot array, size m+1.
mspecifies the size of the knot array.
uis parameter in the [0, 1] interval at which the curve will be evaluated.
Cis an array that receives the 3-D position of the parametric curve C(u) evaluated at "u".

References N3D.

Calculate the control polygon of the derivative of a b-spline curve.

Calculate the control points Q of a b-spline curve that is the dC/du derivative of another b-spline C(u) defined by the control points P. The derivative of any b-spline of degree "p" is another b-spline of degree "p-1", defined by a new set of control points which are calculated here. Reference: http://www.cs.mtu.edu/~shene/COURSES/cs3621/NOTES/spline/B-spline/bspline-derv.html

Calculate the moving triad of a parametric curve C(u) (tangent + normal + binormal) and its curvature.

Calculate the moving triad of a parametric curve C(u) (tangent + normal + binormal) and its curvature, given the first and second order derivatives with respect to the curve parameter dC/du and d2C/du2. The tangent and normal vectors lie in the curvature plane. The binormal is perpendicular to the curvature plane. Reference: http://www.cs.mtu.edu/~shene/COURSES/cs3621/NOTES/curves/normal.html

Parameters
dCduis the first-order derivative of the curve w.r.t its parameter u.
d2Cdu2is the second-order derivative of the curve w.r.t. its parameter u.
Tangentis an array that receives the tangent unit vector.
Normalis an array that receives the normal unit vector.
Binormalis an array that receives the binormal unit vector (normal to the osculating plane).
pCurvatureis a pointer that receives the curvature k = 1/R.
