Detailed Description

The geoid undulation model. This module implements a geoid undulation model using the EGM-2008 geoid model (tide-free) and assuming WGS-84 Earth ellipsoid. The calculator performs a table lookup and a bilinear interpolation using the 2.5 min geoid grid, and returns the altitude of the geoid relative to the ellipsoid surface.
